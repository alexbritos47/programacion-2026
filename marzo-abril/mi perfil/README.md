
# Consigna ejercicio del día 13.3.2026




🟢 Nivel 1 — Mi perfil personal
📌 Consigna
Crear una página web que sea tu perfil personal.
La página debe tener:
Un título en la pestaña del navegador con tu nombre.


Un título principal en la página con tu nombre o apodo.


Un párrafo explicando quién sos.


Un segundo párrafo contando:


qué te gusta hacer


o qué te interesa aprender.


Dentro de uno de los párrafos debe haber un salto de línea .


Agregar una línea separadora .


Debajo de la línea, escribir un último párrafo con algo extra, por ejemplo:


tu comida favorita


tu juego favorito


tu serie favorita.



🟡 Nivel 2 — Mejorando mi perfil
📌 Consigna
Seguir trabajando sobre la misma página y agregar:
Una lista con al menos 3 cosas que te gustan
 (por ejemplo: juegos, comidas, deportes).
 Una imagen que te represente
 (puede ser un hobby, un juego, un animal, etc).
Un enlace a una página que te guste
 (por ejemplo YouTube, un juego, una red social, etc).
Una tabla simple con 2 columnas y 2 filas.
 Ejemplo de ideas:

Algo que pasó

Empecé programación
2025
Aprendí HTML

Un campo para escribir texto
 (por ejemplo: "Dejá un comentario").
Un selector con al menos 3 opciones, por ejemplo:
HTML


CSS


JavaScript

📌 Consigna
Seguir ampliando tu página agregando etiquetas nuevas que no hayas usado.

Al menos 6 etiquetas nuevas.
